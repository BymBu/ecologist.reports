# Ecolog — система для экологической отчётности

Учёт отходов и автоматическая генерация отчётов 2-ТП, 4-ОС для упрощения работы экологам.

## Структура

- `/ecolog-demo` — PWA-демо (офлайн, без бэкенда)
- `/frontend` — фронтенд на Vue (в разработке)
- `/backend` — бэкенд на Laravel (в разработке)
